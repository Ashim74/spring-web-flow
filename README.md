# Spring Web Flow JSP Demo

This project is a minimal Spring Boot application that demonstrates integrating Spring Web Flow 3 with JSP views on Spring Boot 3.2+.

## Prerequisites

* Java 17+
* Maven 3.9+

## Running the application

```bash
mvn spring-boot:run
```

Then open [http://localhost:8080](http://localhost:8080) and click **Start Transfer** to launch the Web Flow wizard.
